# 대물 연어 (오오모노 샤케) 공략

대물 연어를 빠르고 효율적으로 처치하는 것은 아르바이트 성공의 핵심이에요. 각 연어의 기술적인 데이터와 공략 포인트를 확인해 보세요!

## 👾 대물 연어 데이터 요약표

| 명칭 (KR) | HP (체력) | 주요 특징 | 권장 무기군 |
| :--- | :--- | :--- | :--- |
| **폭탄** | 300 (머리) | 팽창 시에만 데미지 | 차저, 슈터 |
| **어깨패드** | - (장갑) | 봄 2개로 파괴 | 모든 아르바이터 |
| **철판몬** | 500 (본체) | 600 데미지 시 스턴 | 연사력 높은 무기 |
| **뱀** | 500 (조종사) | 긴 몸체, 꼬리가 약점 | 슈터, 블래스터 |
| **타워** | 60 (7단) | 연사 무기로 냄비 제거 | 블래스터, 슈터, 붓 |
| **두더지** | 1200 (본체) | 봄 하나로 즉사 (1200) | 모든 아르바이터 |
| **박쥐** | 900 (본체) | 탄환 반사 시 즉사 | 모든 아르바이터 |
| **하시라** | 100 (각각) | 8마리 처치 시 소멸 | 블래스터, 슈터 |
| **다이버** | 1200 (본체) | 고리 도색 시 장갑 파괴 | 모든 아르바이터 |
| **나베부타** | 500 (조종사) | 낙하 유도 후 상단 공격 | 모든 아르바이터 |
| **텟큐** | 1200 | 가장 높은 체력, 해안 상주 | 고DPS 무기, 차저 |
