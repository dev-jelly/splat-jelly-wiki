# 연어의 종류 (샤케노 슈루이)

연어런에는 다양한 특성을 가진 연어들이 등장합니다. 일반 연어부터 강력한 대물 연어, 그리고 거대한 두목 연어까지 각 적의 특성과 공식 명칭, 효율적인 처치법을 확인하세요.

## 적 카테고리

### 1. 일반 연어 (자코 샤케)
가장 기본이 되는 적들입니다. 개별로는 약하지만 무리를 지어 공격해오므로 주의가 필요합니다.
- **[일반 연어 (샤케)](./small-fry/)**: 가장 흔한 기본 연어입니다.
- **[작은 연어 (코자케)](./small-fry/)**: 작고 빠르며 집요하게 공격합니다.
- **[도스코이](./small-fry/)**: 덩치가 크고 체력이 높은 대형 일반 연어입니다.
- **[타마히로이 (빨래판)](./small-fry/)**: 떨어진 황금 연어알을 가로채가는 연어입니다.

### 2. 대물 연어 (오오모노 샤케)
처치 시 '황금 연어알'을 드랍하는 특수한 연어들입니다. 각기 다른 공략법이 필요합니다.
- **[폭탄 (바쿠단)](./boss-salmon/)**: 머리 위의 폭탄을 터뜨려 처치합니다.
- **[어깨패드 (카타팟도)](./boss-salmon/)**: 미사일 런처에 봄을 던져 넣어 파괴합니다.
- **[철판몬 (텟판)](./boss-salmon/)**: 정면을 공격해 멈춘 뒤 뒤쪽의 본체를 공격합니다.
- **[뱀 (헤비)](./boss-salmon/)**: 꼬리 부분의 조종사를 공격하여 처치합니다.
- **[타워 (타와)](./boss-salmon/)**: 냄비 탑을 차례대로 날려버려 처치합니다.
- **[두더지 (모구라)](./boss-salmon/)**: 땅 밑에서 튀어오를 때 봄을 먹여 처치합니다.
- **[박쥐 (코우모리)](./boss-salmon/)**: 발사하는 비구름 탄을 되받아치거나 본체를 공격합니다.
- **[하시라 (기둥)](./boss-salmon/)**: 기둥 위를 빙글빙글 도는 작은 연어들을 모두 처치합니다.
- **[다이버](./boss-salmon/)**: 바닥의 고리를 자신의 잉크로 칠해 뛰어든 다이버를 무력화합니다.
- **[나베부타 (뚜껑)](./boss-salmon/)**: 아래로 내려왔을 때 위로 올라가 조종사를 처치합니다.
- **[텟큐 (빅샷)](./boss-salmon/)**: 해안가에 설치한 포탄 발사기를 사용하는 텟큐를 직접 처치합니다.

### 3. 두목 연어 (오카시라 샤케)
Extra Wave에서 등장하는 거대 보스입니다.
- **[천하장사 (요코즈나)](./boss-salmon/#천하장사)**: 거대한 몸집으로 짓누르거나 점프 공격을 합니다.
- **[임금연어 (타츠)](./boss-salmon/#임금연어)**: 공중을 유영하며 거대한 입에서 폭발 구체를 발사합니다.
- **[죠 (메가로돈)](./boss-salmon/#죠)**: 지면 아래에서 광범위하게 튀어올라 모든 것을 집어삼킵니다.

### 4. 특수 상황 적
- **[금연어 (킨샤케)](../special-situations/)**: 안개나 금연어 찾기 상황에서 등장하는 빛나는 연어입니다.
- **[드로샤케 (진흙연어)](../special-situations/)**: 간헐천에서 솟아오르는 진흙 덩어리 형태의 연어입니다.

## 대물 연어 요약표

| 공식 명칭 | 체력 (HP) | 주요 특징 | 권장 대처법 |
| :--- | :--- | :--- | :--- |
| **폭탄** | 중 | 머리 위 폭탄 생성 | 폭탄 팽창 시 집중 사격 |
| **어깨패드** | 높음 (장갑) | 멀티 미사일 발사 | 양쪽 컨테이너에 봄 투척 |
| **철판몬** | 중 | 전방 가드 및 돌진 | 사격으로 스턴 후 배후 공격 |
| **뱀** | 중 | 긴 몸체로 추격 | 맨 뒤의 조종사 처치 |
| **타워** | 낮음 (다단) | 하이퍼 프레서 공격 | 연사 무기로 냄비 제거 |
| **두더지** | 중 | 지면 아래서 기습 | 튀어오르는 지점에 봄 설치 |
| **박쥐** | 중 | 비구름 탄 발사 | 탄환 반사 또는 뒤집혔을 때 공격 |
| **하시라** | 낮음 | 기둥 설치 및 회전 공격 | 기둥을 타고 올라가 전원 처치 |
| **다이버** | 중 | 고리 생성 후 다이빙 | 고리 내부를 아군 색으로 칠하기 |
| **나베부타** | 중 | 바리어 생성 및 압사 | 낙하 유도 후 위에서 조종사 공격 |
| **텟큐** | 높음 | 충격파 포탄 발사 | 해안가로 이동하여 직접 사격 |
