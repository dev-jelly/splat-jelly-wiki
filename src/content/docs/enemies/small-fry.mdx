# 일반 연어 (자코 샤케)

일반 연어는 연어런의 가장 기본적인 적들입니다. 개별 체력은 낮지만, 대규모로 몰려올 경우 플레이어를 포위하여 위험한 상황을 초래합니다.

## 일반 연어 종류

### 1. 일반 연어 (샤케 / Chum)
가장 흔하게 볼 수 있는 기본 연어입니다. 잉크가 묻은 후라이팬으로 근접 공격을 합니다.
- **대처**: 모든 무기로 쉽게 처치 가능합니다.

### 2. 작은 연어 (코자케 / Smallfry)
몸집이 작고 매우 빠른 연어입니다. 플레이어의 발밑을 집요하게 노리며 공격합니다.
- **대처**: 에임이 낮아야 하며, 연사력이 좋은 무기나 붓, 롤러로 일괄 처리하는 것이 효율적입니다.

### 3. 도스코이 (Cohock)
거구의 대형 일반 연어입니다. 체력이 매우 높으며 공격 시 플레이어를 멀리 밀쳐냅니다.
- **대처**: 화력이 강한 무기나 블래스터, 또는 여러 명의 집중 사격이 필요합니다. '도스코이 대량 발생' 상황에서는 대포를 사용해 처치합니다.

### 4. 타마히로이 (빨래판 / Snatcher)
공격은 하지 않지만, 필드에 떨어진 황금 연어알을 가로채 바다로 가져가는 연어입니다.
- **대처**: 바다로 도망치기 전에 처치하여 알을 되찾아야 합니다. 가끔 컨테이너 근처로 알을 운반해주는 유용한 경로로 이용되기도 합니다.

## 일반 연어 대처법 요약

| 종류 | 위험도 | 주요 특징 | 권장 무기 |
| :--- | :--- | :--- | :--- |
| **샤케** | 낮음 | 물량 공세 | 슈터, 매뉴버 |
| **코자케** | 중 | 빠른 속도, 작은 피격 판정 | 롤러, 슈터, 붓 |
| **도스코이** | 중 | 높은 체력, 넉백 | 블래스터, 슬로셔, 차저 |
| **타마히로이** | 낮음 | 황금 연어알 탈취 | 모든 무기 |

## 성공을 위한 팁
1. **물량 관리**: 일반 연어를 방치하면 대물 연어 공략이 불가능해집니다. 보이는 즉시 정리하세요.
2. **롤러의 역할**: 롤러는 바닥을 밀고 다니며 일반 연어들을 한 번에 정리하는 데 매우 효과적입니다.
3. **위치 선정**: 포위당하지 않도록 항상 주변 시야를 확보하고 퇴로를 칠해두어야 합니다.
