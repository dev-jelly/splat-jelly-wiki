# 특수 상황 가이드

연어런에는 다양한 특수 상황이 발생합니다. 각 상황에 맞는 대처법을 파악하는 것이 승리의 핵심입니다.

## 특수 상황 종류

### 1. 안개 (Fog)
시야가 제한되는 상황입니다.

**특성**
- **시야 제한**: 시야가 매우 제한됨
- **적 등장**: 자코 적 대량 등장
- **위험도**: 중
- **지속 시간**: 웨이브 중간

**대처법**
- **사거리 확보**: 체이서와 블래스터로 사거리 확보
- **포지션 유지**: 중앙 또는 고지 유지
- **사운드 의존**: 발자국 소리로 적 탐지
- **팀 모임**: 한 곳으로 모여서 대처

**추천 무기 조합**
- **체이서**: 원거리 사격으로 시야 제한 극복
- **블래스터**: 폭발로 범위 공격
- **롤러**: 근접 대처
- **슈터**: 안정적인 사격

### 2. 랜드 (Land)
땅 위에서 적이 몰려오는 상황입니다.

**특성**
- **지상 적**: 땅 위에서 적이 몰려옴
- **고립 위험**: 혼자 고립되면 위험
- **위험도**: 높음
- **지속 시간**: 웨이브 중간

**대처법**
- **고지 장악**: 고지 우선 확보
- **빠른 처리**: 롤러와 블래스터로 빠르게 처리
- **팀 모임**: 빠르게 한 곳으로 모이기
- **위치 유지**: 한 곳에서 대처

**추천 무기 조합**
- **롤러**: 빠른 처리에 효과적
- **블래스터**: 폭발로 범위 공격
- **슈터 x2**: 안정적인 사격
- **체이서**: 원거리 대처

### 3. 황금 연어 (Gusher)
황금 연어가 등장하는 상황입니다.

**특성**
- **황금 연어**: 골드 연어 스크랩 대량 드랍
- **특수 공격**: 황금 연어 특수 공격
- **위험도**: 중
- **지속 시간**: 웨이브 중간

**대처법**
- **다층 공격**: 각 층에서 동시에 사격
- **파티클 관리**: 3연속 히트로 효율 증가
- **동시 공격**: 팀원들이 동시에 사격
- **스크랩 수거**: 놓치지 않고 수거

**추천 무기 조합**
- **블래스터**: 높은 화력
- **체이서**: 정확한 사격
- **슈터**: 안정적인 사격
- **롤러**: 빠른 처리

### 4. 그릴 (Grill)

- **구이 요리 처리**: 빠르게 구이 요리 처리
- **거리 유지**: 불길에서 거리 유지
- **팀워크**: 팀원들과 협력

**추천 무기 조합**
