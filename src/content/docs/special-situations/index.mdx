# 특수 상황 (나이트 웨이브) 공략

하늘이 어두워지고 쿠마산의 목소리가 평소와 다르다면? 바로 **특수 상황**이 시작된 거예요! 일반 웨이브와는 완전히 다른 전략이 필요하니, 미리 숙지해서 팀원들과 호흡을 맞춰 보세요.

## 🌙 주요 특수 상황 종류

### 1. 빛나리 (랏슈 / Rush)
*   **특징**: 특정 플레이어에게 빛무리가 붙으며, 미친 듯이 질주하는 연어 떼가 몰려와요.
*   **핵심**: 팀원들이 한곳에 뭉치는 것이 가장 중요해요! 컨테이너 근처 고지대에 모여서 화력을 집중하세요.

### 2. 안개 (키리 / Fog)
*   **특징**: 시야가 매우 제한되어 대물 연어의 위치를 파악하기 힘들어요.
*   **핵심**: 대물 연어를 발견했다면 즉시 **「카몬!」**을 눌러 위치를 공유해 주세요.

### 3. 그릴 발진 (The Griller)
*   **특징**: 거대한 조리 기구 '그릴'이 타겟을 추격해요.
*   **핵심**: 그릴 뒤쪽의 핑크색 꼬리를 사격해서 멈춘 뒤 처치하세요. 타겟이 된 사람은 그릴을 아군이 공격하기 쉬운 경로로 유도해주는 센스가 필요하답니다.

### 4. 하코비야 습격 (The Mothership)
*   **특징**: 거대 모선이 나타나 알을 빼앗으려 하고, 연어콥터들이 박스를 떨어트려요.
*   **핵심**: 하늘을 나는 연어콥터를 격추해 알을 모으고, 모선이 컨테이너에 접근하면 전원이 집중 사격해서 쫓아내야 해요!

## 🌊 수치로 보는 특수 상황 (데이터)

일본 위키의 기술적인 데이터를 참고해 보았어요.

*   **빛나리**: 킨샤케(금연어)의 HP는 **500**이며, 처치 시 황금 연어알 3개를 드랍해요.
*   **그릴**: 본체 HP는 **2200**이며, 처치 시 황금 연어알 5개를 준답니다.
*   **하코비야**: 모선 본체에 일정 데미지를 주면 퇴각해요. 연어콥터는 HP가 **50**으로 매우 낮으니 연사 무기가 유리해요.
