# 특수 상황 (나이트 웨이브)

연어런에서는 특정 확률로 배경이 어두워지며 평소와 다른 **특수 상황**이 발생합니다. 각 상황은 고유한 규칙을 가지고 있으며, 이를 정확히 파악하는 것이 고득점의 핵심입니다.

## 특수 상황 종류

### 1. 빛나리 (랏슈 / Rush)
특정 플레이어 주변에 '빛나리'가 붙으며, 미친 듯이 질주하는 연어 무리가 해당 플레이어를 집중 공격합니다.
- **특징**: 매우 빠른 속도의 연어 무리와 금연어가 등장합니다.
- **대처**: 팀원이 한곳에 뭉쳐 화력을 집중해야 합니다. 빛나리가 붙은 플레이어는 아군이 사격하기 좋은 위치에 자리를 잡으세요.

### 2. 안개 (키리 / Fog)
스테이지 전체에 짙은 안개가 끼어 시야가 극도로 제한됩니다.
- **특징**: 대물 연어가 어디서 나타날지 알기 어렵고, 금연어가 무작위로 등장합니다.
- **대처**: '카몬(이쪽이다!)' 기능을 활용해 대물 연어의 위치를 적극적으로 공유해야 합니다.

### 3. 그릴 발진 (구리루 핫신 / The Griller)
잉크를 뿌리며 이동하는 거대 조리 기구 '그릴'이 플레이어를 추격합니다.
- **특징**: 그릴은 타겟팅된 플레이어를 끝까지 쫓으며, 경로에 강력한 잉크를 뿌립니다.
- **대처**: 그릴 뒤쪽의 꼬리 부분을 공격해 스턴 상태로 만든 뒤 처치합니다. 타겟이 된 플레이어는 그릴의 유도 방향을 조절하여 아군이 공격하기 쉽게 도와야 합니다.

### 4. 하코비야 습격 (하코비야 슈라이 / The Mothership)
거대 모선 '하코비야'가 나타나 연어콥터를 사출하며 황금 연어알을 빼앗으려 합니다.
- **특징**: 하늘에서 박스가 떨어지며 일반 연어들이 계속 생성됩니다.
- **대처**: 날아다니는 연어콥터를 격추해 황금 연어알을 확보하고, 모선이 컨테이너에 접근할 때 집중 사격하여 쫓아내야 합니다.

### 5. 금연어 찾기 (킨샤케 사가시 / Goldie Seeking)
스테이지의 간헐천 중 하나에 숨어있는 금연어를 찾아야 합니다.
- **특징**: 간헐천을 열어 빛의 세기로 금연어의 위치를 추측합니다.
- **대처**: 금연어를 발견하면 화력을 집중해 최대한 많은 황금 연어알을 떨어뜨리게 합니다. 불필요하게 간헐천을 많이 열면 일반 연어만 늘어나므로 주의가 필요합니다.

### 6. 도스코이 대량 발생 (도스코이 타이료 하세이 / Cohock Charge)
간조 시에만 발생하며, 거구의 도스코이들이 떼를 지어 몰려옵니다.
- **특징**: 플레이어의 무기만으로는 처리가 힘들 정도로 많은 도스코이가 등장합니다.
- **대처**: 해안가에 설치된 캐논에 탑승한 팀원이 도스코이 무리를 포격으로 정리하고, 나머지 팀원은 황금 연어알을 신속히 납품합니다.

### 7. 드로샤케 분출 (드로샤케 훈슈츠 / Mudmouth Eruptions)
간헐천에서 진흙으로 된 연어 '드로샤케'가 솟아오릅니다.
- **특징**: 입 안으로 일반 연어나 꼬마 연어를 계속 뱉어냅니다.
- **대처**: 드로샤케의 입 안으로 **스플래시 봄**을 던져 넣어 처치합니다. 황금색 드로샤케는 더 많은 알을 드랍합니다.

### 8. 거대 타츠마키 (쿄다이 타츠마키 / Giant Tornado)
거대한 토네이도가 발생하여 바다에서 황금 연어알 상자가 날아옵니다.
- **특징**: 컨테이너가 멀리 떨어져 있고 일반 연어들이 하늘에서 떨어집니다.
- **대처**: 팀원들이 일렬로 서서 '연어알 던지기' 릴레이를 통해 알을 컨테이너까지 신속하게 운반해야 합니다.

## 공통 전략 가이드

1. **상황 판단**: WAVE 시작 시 배경이 어두워지고 쿠마산의 대사가 바뀌면 즉시 특수 상황임을 인지하세요.
2. **위치 선정**: 대부분의 특수 상황은 팀워크가 중요하므로, 고립되지 말고 컨테이너 주변이나 유리한 지점에 모여야 합니다.
3. **스페셜 사용**: 특수 상황에서 전멸 위기에 처하면 아끼지 말고 스페셜 웨폰을 사용하여 상황을 반전시키세요.
