# 무기 및 기어 (부키·기아)

연어런에서는 자신의 개인 무기를 사용할 수 없으며, 쿠마산 상회에서 지급하는 전용 무기와 작업복을 사용해야 합니다. 각 무기군의 특성을 이해하고 팀의 밸런스에 맞춰 행동하는 것이 중요합니다.

## 무기 카테고리

### 1. 슈터 / 매뉴버
가장 균형 잡힌 무기군입니다. 일반 연어 처리와 칠하기에 능숙합니다.

### 2. 차저 / 스피너
긴 사거리와 압도적인 화력을 자랑합니다. 대물 연어나 두목 연어를 상대하기 좋습니다.

### 3. 블래스터 / 슬로셔
광역 데미지가 특징입니다. 타워나 하시라 처리에 효과적입니다.

### 4. 롤러 / 붓 / 와이퍼
근접 전투와 물량 정리에 최적화되어 있습니다.
