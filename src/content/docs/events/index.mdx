# 이벤트 (이벤토)

연어런 NEXT WAVE에서는 정기적으로 특별한 규칙과 보상이 제공되는 이벤트가 개최됩니다. 각 이벤트의 특성을 파악하여 한정 보상을 획득하세요.

## 주요 이벤트 종류

### 1. 빅 런 (빅구 란 / Big Run)
수개월에 한 번씩 발생하는 대규모 이벤트로, 연어들이 도심지(일반 배틀용 스테이지)까지 침공해오는 상황입니다.
- **특징**: 평소에는 플레이할 수 없는 일반 맵에서 연어런을 진행합니다.
- **보상**: 상위 점수 기록에 따라 특별한 '두목 연어 장식품'과 배지를 획득할 수 있습니다.

### 2. 바이트 팀 콘테스트 (Team Contest)
고정된 시나리오(무기, 연어 등장 순서 등)에서 팀원들과 협력하여 높은 점수를 겨루는 대회형 이벤트입니다.
- **특징**: 모든 상황이 고정되어 있으므로, 반복 플레이를 통해 최적의 빌드를 짜는 것이 중요합니다.
