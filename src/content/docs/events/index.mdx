# 이벤트 가이드

연어런에는 다양한 이벤트가 있습니다. 각 이벤트의 특성과 참여 방법을 파악하여 효율적으로 참여하세요.

## 이벤트 종류

### 1. 빅 런 (Big Run)
도시에서 연어런을 하는 이벤트입니다.

**특성**
- **도시 배경**: 도시에서 연어런 진행
- **거대 맵**: 일반 스테이지보다 3-4배 큰 맵
- **다양한 환경**: 해변, 건물, 공원 등
- **보상**: 특별한 보상 제공

**참여 방법**
- **이벤트 기간**: 이벤트 기간 확인
- **참여 조건**: 참여 조건 확인
- **맵 학습**: 거대 맵 학습
- **전략**: 거대 맵에 맞는 전략

**전략**
- **맵 학습**: 넓은 지형의 특징 파악
- **빠른 이동**: 넓은 맵에서의 효율적 이동
- **구역 분담**: 각 구역별 담당 정하기
- **상황 공유**: 넓은 맵에서의 커뮤니케이션

**보상**
- **특별 보상**: 이벤트 특별 보상
- **경험치**: 높은 경험치
- **스크랩**: 많은 스크랩
- **배지**: 특별 배지

### 2. 팀 컨테스트 (Team Contest)
팀 대항전 이벤트입니다.

**특성**
- **팀 대항**: 팀 간 대항전
- **랭킹**: 팀 랭킹 시스템
- **협력 플레이**: 팀 협력 플레이
- **보상**: 팀 성과에 따른 보상

**참여 방법**
- **팀 구성**: 팀 구성
- **이벤트 기간**: 이벤트 기간 확인
- **참여 조건**: 참여 조건 확인
- **전략**: 팀 전략 수립

**전략**
- **팀워크**: 팀원들과 협력
- **랭킹 관리**: 팀 랭킹 관리
- **무기 조합**: 팀 무기 조합 고려
- **커뮤니케이션**: 팀 커뮤니케이션

**보상**
- **팀 보상**: 팀 성과에 따른 보상
- **개인 보상**: 개인 성과에 따른 보상
- **배지**: 특별 배지
- **경험치**: 높은 경험치
