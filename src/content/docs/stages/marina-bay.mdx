# 마리나 스퀘어 (Marina Bay)

![Marina Bay](/images/stages/marina-bay.jpg)

## 스테이지 개요

마리나 스퀘어는 해변 상업 지구를 배경으로 한 연어런 스테이지입니다. 수평 구조와 넓은 플랫폼이 특징이며, 쇼핑몰과 편의점 "나우(NOW)" 사이의 구역을 무대로 합니다.

## 지형 특징

### 메인 구역
- **하부 플랫폼**: 바스켓이 위치한 메인 구역
- **중앙 구역**: 넓은 중앙 플랫폼
- **우측 상부**: 높은 위치의 발판
- **좌측 경로**: 수평으로 이어지는 경로

### 주요 지형 요소
1. **중앙 플랫폼**: 넓은 개방된 구역
2. **우측 상부**: 높은 위치, 우수한 사격 포지션
3. **좌측 경로**: 긴 수평 경로
4. **하부 플랫폼**: 바스켓과 하부 구역

## 전략 포인트

### 웨이브 1: 기본 웨이브
- **포지셔닝**: 중앙 2명, 우측 상부 1명, 좌측 1명
- **추천 구성**: 슈터 x2 + 블래스터 + 롤러
- **주의사항**: 넓은 플랫폼에서의 적 접근 대비

### 웨이브 2: 난이도 상승
- **우측 상부 장악**: 높은 위치 우선 확보
- **중앙 방어**: 개방된 구역에서 적 막아내기
- **협력 플레이**: 상부와 하부 연계

### 웨이브 3: 라이드 레일
- **이동 경로**: 하부 → 중앙 → 우측 상부
- **골드 연어**: 상부와 중앙에서 수거
- **전략**: 넓은 지형에서의 빠른 이동

### 웨이브 X: 보스 러시
- **진행 경로**: 하부 → 중앙 → 상부
- **보스 대처**: 상부에서 보스 처리
- **팀워크**: 레일 위에서의 협력 어택

## 적 등장 패턴

### 자코 적
- **하부 등장**: 하부 플랫폼에서 가장 빈번
- **중앙 등장**: 넓은 중앙 구역으로 이동
- **상부 등장**: 우측 상부에서 공격

### 보스 연어
1. **배틀**: 하부 플랫폼에서 주로 등장
2. **탕크**: 중앙 플랫폼 주변
3. **카브**: 우측 상부로 등반
4. **스타팅**: 우측 상부에서 지원 사격
5. **샤크**: 하부에서 급습
6. **그리드**: 중앙 포위
7. **어뢰**: 하부 구역 침공

## 추천 무기 조합

### 밸런스 조합
- **슈터**: 샤크 스탠다드
- **체이서**: 하이퍼 레드
- **롤러**: 와이드
- **블래스터**: 큐어

### 특수 상황 대응
- **안개**: 체이서 + 블래스터 + 롤러 + 슈터
- **랜드**: 롤러 + 블래스터 + 슈터 x2
- **황금 연어**: 블래스터 + 체이서 + 슈터 + 롤러

## 고급 전략

### 우측 상부 활용
- **사격 포지션**: 중앙과 하부 모두 공격 가능
- **골드 연어 수거**: 상부로 낙하하는 연어
- **회피**: 하강하며 적 회피

### 중앙 플랫폼 활용
- **중앙 장악**: 스테이지의 핵심 위치
- **연계 공격**: 상부와 하부 연계
- **넓은 시야**: 주변 상황 파악 용이

### 좌측 경로 활용
- **수평 이동**: 긴 경로로의 빠른 이동
- **적 처리**: 좌측에서 나타나는 적 대응
- **높낮이 이용**: 경로에서 상부 적 공격

### 골드 연어 수거
- **하부 낙하**: 상부에서 떨어지는 연어
- **중앙 수거**: 중앙 플랫폼에 떨어지는 연어
- **이동 중 수거**: 레일 위에서 수거

### 팀워크
- **하부 담당**: 바스켓 보호 및 하부 적 처리
- **중앙 담당**: 중앙에서 사격 및 지원
- **상부 담당**: 우측 상부에서 사격 지원

## 특수 상황 대처법

### 안개 (Fog)
- **사거리 확보**: 체이서와 블래스터 활용
- **포지션 유지**: 중앙 플랫폼 집중
- **사운드 의존**: 발자국 소리로 적 탐지

### 랜드 (Land)
- **상부 장악**: 우측 상부 우선 확보
- **중앙 방어**: 넓은 구역에서 대응
- **팀 모집**: 빠르게 중앙으로 집결

### 황금 연어 (Gusher)
- **다층 공격**: 각 층에서 동시에 사격
- **파티클 관리**: 3연속 히트로 효율 증가
- **동시 공격**: 팀원들이 동시에 사격

## 성공을 위한 팁

1. **중앙 플랫폼 장악**: 스테이지의 핵심 위치
2. **우측 상부 활용**: 높은 위치 우위 확보
3. **넓은 시야 활용**: 개방된 지형에서의 상황 파악
4. **팀 분배**: 하부/중앙/상부 균등 분배
5. **골드 연어 수거**: 상부에서 떨어지는 연어 놓치지 않기
6. **보스 처리 우선**: 하부에서 스크랩 놓침 방지
7. **커뮤니케이션**: 상황 공유 및 협력 플레이

## 주의사항

- **상부 오래 머무르지 않기**: 적에게 둘러싸일 위험
- **넓은 지형 주의**: 여러 방향에서 적 접근
- **혼자 고립되지 않기**: 팀원과 함께 이동
- **골드 연어 놓치지 않기**: 상부에서 낙하 확인
- **보스 공략 순서**: 하부 위험도 높은 보스부터
