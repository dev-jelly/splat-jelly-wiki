# 맛조개 방수로 (Undertow Spillway) 공략

마테가이 방수로는 길게 뻗은 직선 통로와 복잡한 고저차가 특징인 빅 런 스테이지예요. 좁은 길목에서 화력을 얼마나 집중하느냐가 생존의 핵심이랍니다.

## 🗺️ 지형 특징 및 전략
*   **직선 도로 사수**: 통로가 좁아 연어들이 일렬로 몰려와요. 관통 무기가 있다면 이곳에서 엄청난 효율을 낼 수 있어요.
*   **유리벽 활용**: 사격은 통하지만 이동은 불가능한 유리벽 너머의 아군을 엄호해 주세요.
