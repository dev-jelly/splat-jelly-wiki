# 뉴 아메리칸 다운 (New Albacore Hotel)

![New Albacore Hotel](/images/stages/new-albacore-hotel.jpg)

## 스테이지 개요

뉴 아메리칸 다운은 호텔 건물을 배경으로 한 연어런 스테이지입니다. 수직 구조와 다층 플랫폼이 특징이며, 높낮이 차이가 큰 지형을 가지고 있습니다.

## 지형 특징

### 메인 구역
- **하부 로비**: 바스켓이 위치한 메인 구역
- **중층 발판**: 여러 개의 중간 높이 발판
- **상부 옥상**: 건물 꼭대기 전망
- **우측 경로**: 외부로 이어지는 계단

### 주요 지형 요소
1. **중층 발판**: 여러 개로 분리된 플랫폼
2. **상부 옥상**: 가장 높은 위치, 전체 조망
3. **우측 계단**: 외부로 이어지는 경로
4. **엘리베이터**: 상하 이동을 돕는 구조물

## 전략 포인트

### 웨이브 1: 기본 웨이브
- **포지셔닝**: 하부 2명, 중층 1명, 상부 1명
- **추천 구성**: 슈터 + 블래스터 + 롤러 + 체이서
- **주의사항**: 높낮이 차이가 큰 지형에서의 이동

### 웨이브 2: 난이도 상승
- **높낮이 활용**: 각 층에서 적 처리
- **빠른 이동**: 엘리베이터 활용
- **협력 플레이**: 각 층 연계

### 웨이브 3: 라이드 레일
- **이동 경로**: 로비 → 중층 발판 → 상부 옥상
- **골드 연어**: 옥상과 중층에서 수거
- **전략**: 수직 이동과 정확한 사격

### 웨이브 X: 보스 러시
- **진행 경로**: 로비 → 중층 → 옥상
- **보스 대처**: 옥상에서 보스 처리
- **팀워크**: 레일 위에서의 협력 어택

## 적 등장 패턴

### 자코 적
- **하부 등장**: 로비에서 가장 빈번
- **중층 등장**: 발판으로 이동
- **상부 등장**: 옥상에서 공격

### 보스 연어
1. **배틀**: 로비에서 주로 등장
2. **탕크**: 중층 발판 주변
3. **카브**: 엘리베이터 주변 등반
4. **스타팅**: 상부 옥상에서 지원 사격
5. **샤크**: 로비에서 급습
6. **그리드**: 발판 포위
7. **어뢰**: 로비 구역 침공

## 추천 무기 조합

### 밸런스 조합
- **슈터**: 샤크 스탠다드
- **체이서**: 하이퍼 레드
- **롤러**: 다이나모
- **블래스터**: 로켓 피스톨

### 특수 상황 대응
- **안개**: 체이서 + 블래스터 + 롤러 + 슈터
- **랜드**: 롤러 + 블래스터 + 슈터 x2
- **황금 연어**: 블래스터 + 체이서 + 슈터 + 롤러

## 고급 전략

### 상부 옥상 활용
- **사격 포지션**: 전체 지형을 내려다보며 사격
- **골드 연어 수거**: 옥상으로 낙하하는 연어
- **회피**: 하강하며 적 회피

### 중층 발판 활용
- **분산 대응**: 여러 발판에서 적 처리
- **연계 공격**: 발판 간 협력
- **높낮이 이용**: 중층에서 하부 적 공격

### 골드 연어 수거
- **하부 낙하**: 옥상에서 떨어지는 연어
- **중층 수거**: 발판 위로 낙하하는 연어
- **이동 중 수거**: 레일 위에서 수거

### 팀워크
- **하부 담당**: 바스켓 보호 및 로비 적 처리
- **중층 담당**: 발판에서 사격 및 지원
- **상부 담당**: 옥상에서 사격 지원

## 특수 상황 대처법

### 안개 (Fog)
- **사거리 확보**: 체이서와 블래스터 활용
- **포지션 유지**: 상부 옥상 집중
- **사운드 의존**: 발자국 소리로 적 탐지

### 랜드 (Land)
- **옥상 장악**: 상부 옥상 우선 확보
- **엘리베이터 활용**: 빠른 이동
- **팀 모집**: 빠르게 옥상으로 집결

### 황금 연어 (Gusher)
- **다층 공격**: 각 층에서 동시에 사격
- **파티클 관리**: 3연속 히트로 효율 증가
- **동시 공격**: 팀원들이 동시에 사격

## 성공을 위한 팁

1. **상부 옥상 장악**: 스테이지의 핵심 위치
2. **높낮이 활용**: 각 층의 장점 활용
3. **빠른 이동**: 엘리베이터 효율적 활용
4. **팀 분배**: 하부/중층/상부 균등 분배
5. **골드 연어 수거**: 옥상에서 떨어지는 연어 놓치지 않기
6. **보스 처리 우선**: 로비에서 스크랩 놓침 방지
7. **커뮤니케이션**: 상황 공유 및 협력 플레이

## 주의사항

- **옥상 오래 머무르지 않기**: 적에게 둘러싸일 위험
- **높낮이 주의**: 이동 중 추락 주의
- **혼자 고립되지 않기**: 팀원과 함께 이동
- **골드 연어 놓치지 않기**: 옥상에서 낙하 확인
- **보스 공략 순서**: 로비 위험도 높은 보스부터
