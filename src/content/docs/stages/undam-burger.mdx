# 이루나 뷰 (Undam Burger)

![Undam Burger](/images/stages/undam-burger.jpg)

## 스테이지 개요

이루나 뷰는 두 번째 빅 런 이벤트 스테이지입니다. 햄버거 가게 "이루나 뷰"를 배경으로 하며, 가게 건물, 주차장, 로드 등 다양한 지형을 가지고 있습니다.

## 지형 특징

### 메인 구역
- **주차장**: 바스켓이 위치한 주차장
- **가게 건물**: 이루나 뷰 햄버거 가게
- **상부 발판**: 건물 위의 높은 발판
- **도로**: 가게 앞에 위치한 도로

### 주요 지형 요소
1. **주차장**: 시작 지점, 넓은 공간
2. **가게 건물**: 이루나 뷰 가게 건물
3. **상부 발판**: 건물 위 발판, 우수한 사각 포지션
4. **도로**: 가게 앞 도로 지역

## 빅 런 특징

### 일반 연어런과의 차이점
- **거대 지형**: 일반 스테이지보다 3-4배 큰 맵
- **다양한 환경**: 주차장, 가게, 도로 등
- **복잡한 경로**: 여러 루트로 이동 가능
- **추가 목표**: 연어 스크랩 외에 추가 미션 존재

### 웨이브 구성
- **웨이브 1-3**: 기본 빅 런 웨이브
- **목표**: 일반 연어런과 동일한 스크랩 수거
- **보스 연어**: 일반 연어런과 유사하지만 더 많이 등장

## 전략 포인트

### 포지셔닝
- **주차장 담당**: 주차장에서 적 처리
- **가게 담당**: 가게 건물 내에서 적 처리
- **상부 담당**: 상부 발판에서 사격 지원
- **도로 담당**: 도로 근처에서 적 처리

### 빅 런 특수 전략
- **맵 학습**: 넓은 지형 파악
- **분산 방어**: 여러 구역으로 분산
- **빠른 이동**: 넓은 맵에서의 빠른 이동
- **팀워크**: 구역 간 협력

### 라이드 레일
- **이동 경로**: 주차장 → 가게 → 상부 → 도로
- **골드 연어**: 각 구역에서 수거
- **전략**: 넓은 지형에서의 레일 활용

## 적 등장 패턴

### 자코 적
- **주차장 등장**: 주차장에서 가장 빈번
- **가게 등장**: 가게 건물 내부로 이동
- **상부 등장**: 상부 발판에서 공격
- **도로 등장**: 도로 근처에서 등장

### 보스 연어
1. **배틀**: 주차장과 가게에서 주로 등장
2. **탕크**: 가게 주변과 상부
3. **카ブ**: 가게로 등반
4. **스타팅**: 상부 발판에서 지원 사격
5. **샤크**: 주차장과 도로에서 급습
6. **그리드**: 가게 포위
7. **어뢰**: 주차장 구역 침공

## 추천 무기 조합

### 밸런스 조합
- **슈터**: 샤크 스탠다드
- **체이서**: 하이퍼 레드
- **롤러**: 다이나모
- **블래스터**: 로켓 피스톨

### 빅 런 대응
- **맵 활용**: 넓은 지형에서의 사거리 확보
- **이동성**: 빠른 이동 가능한 무기
- **협력 플레이**: 다양한 무기 조합

## 고급 전략

### 넓은 지형 활용
- **맵 학습**: 각 구역의 특징 파악
- **빠른 이동**: 넓은 맵에서의 효율적 이동
- **분산 방어**: 여러 구역으로 분산

### 가게 건물 활용
- **내부 공격**: 가게 내부에서 적 처리
- **상부 이용**: 가게 위에서 사격
- **회피 경로**: 가게를 이용한 회피

### 도로 활용
- **도로 구역**: 도로 근처 적 처리
- **넓은 시야**: 도로에서의 상황 파악
- **팀워크**: 도로 주변 협력

### 골드 연어 수거
- **각 구역 수거**: 주차장, 가게, 도로에서 수거
- **이동 중 수거**: 레일 위에서 수거
- **협력 수거**: 팀원들과 분산 수거

### 팀워크
- **구역 분담**: 각 구역별 담당 정하기
- **상황 공유**: 넓은 맵에서의 상황 공유
- **빠른 지원**: 필요한 곳으로 빠르게 이동

## 특수 상황 대처법

### 안개 (Fog)
- **사거리 확보**: 체이서와 블래스터 활용
- **포지션 유지**: 가게 주변 집중
- **사운드 의존**: 발자국 소리로 적 탐지

### 랜드 (Land)
- **상부 장악**: 가게 위 우선 확보
- **가게 방어**: 가게 내부에서 대응
- **팀 모집**: 빠르게 한 곳으로 집결

### 황금 연어 (Gusher)
- **다층 공격**: 각 구역에서 동시에 사격
- **파티클 관리**: 3연속 히트로 효율 증가
- **동시 공격**: 팀원들이 동시에 사격

## 성공을 위한 팁

1. **맵 학습**: 넓은 지형의 특징 파악
2. **빠른 이동**: 넓은 맵에서의 효율적 이동
3. **구역 분담**: 각 구역별 담당 정하기
4. **상황 공유**: 넓은 맵에서의 커뮤니케이션
5. **골드 연어 수거**: 각 구역에서 수거
6. **보스 처리 우선**: 넓은 맵에서의 스크랩 놓침 방지
7. **팀워크**: 구역 간 협력 플레이

## 주의사항

- **넓은 맵**: 긴 이동 거리에 주의
- **혼자 고립되지 않기**: 팀원과 함께 이동
- **골드 연어 놓치지 않기**: 넓은 맵에서 수거 확인
- **상황 파악**: 넓은 맵에서의 상황 인식
- **빠른 대응**: 필요한 곳으로 빠르게 이동
