파밀리 벤드

파밀리 벤드 (Family’s Bend)

스테이지 개요

파밀리 벤드는 여섯 번째 빅 런 이벤트 스테이지입니다. 주거 지역을 배경으로 하며, 주택, 공원, 상가 등 다양한 지형을 가지고 있습니다.

지형 특징

메인 구역

  • 주택가: 바스켓이 위치한 주택가
  • 공원: 공원 구역
  • 상가: 상가 지역
  • 상부 발판: 건물 위의 높은 발판

주요 지형 요소

  1. 주택가: 시작 지점, 넓은 공간
  2. 공원: 공원 구역
  3. 상가: 상가 건물들
  4. 상부 발판: 건물 위 발판, 우수한 사각 포지션

빅 런 특징

일반 연어런과의 차이점

  • 거대 지형: 일반 스테이지보다 3-4배 큰 맵
  • 다양한 환경: 주택, 공원, 상가 등
  • 복잡한 경로: 여러 루트로 이동 가능
  • 추가 목표: 연어 스크랩 외에 추가 미션 존재

웨이브 구성

  • 웨이브 1-3: 기본 빅 런 웨이브
  • 목표: 일반 연어런과 동일한 스크랩 수거
  • 보스 연어: 일반 연어런과 유사하지만 더 많이 등장

전략 포인트

포지셔닝

  • 주택가 담당: 주택가에서 적 처리
  • 공원 담당: 공원 구역에서 적 처리
  • 상가 담당: 상가 지역에서 적 처리
  • 상부 담당: 상부 발판에서 사격 지원

빅 런 특수 전략

  • 맵 학습: 넓은 지형 파악
  • 분산 방어: 여러 구역으로 분산
  • 빠른 이동: 넓은 맵에서의 빠른 이동
  • 팀워크: 구역 간 협력

라이드 레일

  • 이동 경로: 주택가 → 공원 → 상가 → 상부
  • 골드 연어: 각 구역에서 수거
  • 전략: 넓은 지형에서의 레일 활용

적 등장 패턴

자코 적

  • 주택가 등장: 주택가에서 가장 빈번
  • 공원 등장: 공원 구역으로 이동
  • 상가 등장: 상가 지역에서 공격
  • 상부 등장: 상부 발판에서 등장

보스 연어

  1. 배틀: 주택가와 공원에서 주로 등장
  2. 탕크: 공원 주변과 상부
  3. 카브: 건물로 등반
  4. 스타팅: 상부 발판에서 지원 사격
  5. 샤크: 주택가에서 급습
  6. 그리드: 공원 포위
  7. 어뢰: 주택가 구역 침공

추천 무기 조합

밸런스 조합

  • 슈터: 샤크 스탠다드
  • 체이서: 하이퍼 레드
  • 롤러: 다이나모
  • 블래스터: 로켓 피스톨

빅 런 대응

  • 맵 활용: 넓은 지형에서의 사거리 확보
  • 이동성: 빠른 이동 가능한 무기
  • 협력 플레이: 다양한 무기 조합

고급 전략

넓은 지형 활용

  • 맵 학습: 각 구역의 특징 파악
  • 빠른 이동: 넓은 맵에서의 효율적 이동
  • 분산 방어: 여러 구역으로 분산

주택가 활용

  • 주택 공격: 주택가에서 적 처리
  • 건물 활용: 주택 건물을 이용한 회피
  • 높낮이 이용: 주택가 지형의 특성 활용

공원 활용

  • 공원 구역: 공원에서 적 처리
  • 지형 활용: 공원 구조물 활용
  • 팀워크: 공원 주변 협력

골드 연어 수거

  • 각 구역 수거: 주택가, 공원, 상가에서 수거
  • 이동 중 수거: 레일 위에서 수거
  • 협력 수거: 팀원들과 분산 수거

팀워크

  • 구역 분담: 각 구역별 담당 정하기
  • 상황 공유: 넓은 맵에서의 상황 공유
  • 빠른 지원: 필요한 곳으로 빠르게 이동

특수 상황 대처법

안개 (Fog)

  • 사거리 확보: 체이서와 블래스터 활용
  • 포지션 유지: 공원 주변 집중
  • 사운드 의존: 발자국 소리로 적 탐지

랜드 (Land)

  • 상부 장악: 상부 발판 우선 확보
  • 주택가 방어: 주택가에서 대응
  • 팀 모집: 빠르게 한 곳으로 집결

황금 연어 (Gusher)

  • 다층 공격: 각 구역에서 동시에 사격
  • 파티클 관리: 3연속 히트로 효율 증가
  • 동시 공격: 팀원들이 동시에 사격

성공을 위한 팁

  1. 맵 학습: 넓은 지형의 특징 파악
  2. 빠른 이동: 넓은 맵에서의 효율적 이동
  3. 구역 분담: 각 구역별 담당 정하기
  4. 상황 공유: 넓은 맵에서의 커뮤니케이션
  5. 골드 연어 수거: 각 구역에서 수거
  6. 보스 처리 우선: 넓은 맵에서의 스크랩 놓침 방지
  7. 팀워크: 구역 간 협력 플레이

주의사항

  • 넓은 맵: 긴 이동 거리에 주의
  • 혼자 고립되지 않기: 팀원과 함께 이동
  • 골드 연어 놓치지 않기: 넓은 맵에서 수거 확인
  • 상황 파악: 넓은 맵에서의 상황 인식
  • 빠른 대응: 필요한 곳으로 빠르게 이동
