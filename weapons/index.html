무기

연어런 무기

연어런에서는 다양한 무기를 사용할 수 있습니다. 각 무기는 고유한 특성을 가지고 있으며, 적절한 무기 조합이 승리의 핵심입니다.

무기 카테고리

1. 슈터 (Shooters)

기본적인 사격 무기입니다.

  • 샤크 스탠다드 (Shark Standard)
  • 쿠로스 (Kuros)
  • 스펙트럼 (Spectrum)

2. 체이서 (Chargers)

충전 사격 무기입니다.

  • 하이퍼 레드 (Hyper Red)
  • 호버 레드 (Hover Red)

3. 롤러 (Rollers)

롤링과 털치기 무기입니다.

  • 다이나모 (Dynamo)
  • 와이드 (Wide)

4. 블래스터 (Blasters)

폭발형 무기입니다.

  • 로켓 피스톨 (Rocket Piston)
  • 큐어 (Cure)

5. 특수 무기 (Special Weapons)

특수 상황에서 사용하는 무기입니다.

  • 로보 아머 (Robo Armor)
  • 크레이터 (Crater)
  • 슈퍼 점프 (Super Jump)
  • 잉크 스톰 (Ink Storm)
  • 로보 보이 (Robo Boy)
  • 파이프 스톰 (Pipe Storm)
  • 로보 망치 (Robo Hammer)
  • 슈퍼 체이서 (Super Charger)

무기 카테고리별 특성

슈터 (Shooters)

  • 사거리: 중
  • 화력: 중
  • 사용 용이성: 높음
  • 특성: 안정적인 사격, 범용성 높음

체이서 (Chargers)

  • 사거리: 긴
  • 화력: 높음
  • 사용 용이성: 중
  • 특성: 원거리 강력한 사격, 충전 필요

롤러 (Rollers)

  • 사거리: 짧음
  • 화력: 높음
  • 사용 용이성: 중
  • 특성: 광역 처리, 근접 강력

블래스터 (Blasters)

  • 사거리: 중
  • 화력: 높음
  • 사용 용이성: 중
  • 특성: 폭발 대처, 광역 공격

특수 무기 (Special Weapons)

  • 사거리: 다양
  • 화력: 다양
  • 사용 용이성: 중
  • 특성: 특수 상황 강력

무기 조합

밸런스 조합

  • 슈터 x2: 안정적인 사격
  • 체이서 x1: 원거리 대처
  • 롤러 x1: 광역 처리
  • 블래스터 x1: 폭발 대처

공격적 조합

  • 슈터 x2: 안정적인 사격
  • 블래스터 x2: 폭발 대처 강화
  • 롤러 x1: 광역 처리
  • 체이서 x0: 원거리 약화

방어적 조합

  • 체이서 x2: 원거리 강화
  • 롤러 x1: 광역 처리
  • 블래스터 x1: 폭발 대처
  • 슈터 x1: 안정적인 사격

특수 상황 조합

  • 안개: 체이서 + 블래스터 + 롤러 + 슈터
  • 랜드: 롤러 + 블래스터 + 슈터 x2
  • 황금 연어: 블래스터 + 체이서 + 슈터 + 롤러

일반적인 전략

무기 선택

  1. 자신의 장점: 자신의 장점에 맞는 무기 선택
  2. 팀 밸런스: 팀 무기 조합 고려
  3. 스테이지: 스테이지 특성 고려
  4. 상황: 상황에 맞는 무기 선택

무기 사용

  1. 거리 유지: 적절한 거리 유지
  2. 사격 타이밍: 적절한 사격 타이밍
  3. 특수 무기: 특수 무기 적절히 사용
  4. 팀워크: 팀원들과 협력
