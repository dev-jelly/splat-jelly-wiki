# 스플래툰 3 연어런 용어 사전 v2 (비공식 가이드용)

본 가이드는 스플래툰 3의 '연어런 NEXT WAVE'를 위한 비공식 공략 사이트입니다. 모든 게임 콘텐츠의 저작권은 **Nintendo**에 있습니다.

## 🎯 용어 및 데이터 원칙
1. **공식 명칭 최우선**: 한국닌텐도 공식 로컬라이징 명칭을 1순위로 사용합니다.
2. **커뮤니티 용어 보완**: 공식 명칭이 어색하거나(예: 임금연어), 커뮤니티에서 더 직관적으로 사용되는 용어(예: 용)를 적극 권장합니다.
3. **데이터 단위**: 
   - **프레임(F)**: 60fps 기준입니다. 
   - **초(sec)**: 프레임 수치를 60으로 나눈 초 단위 데이터를 병기합니다 (예: 60F = 1.0초).
4. **일본어 독음 최소화**: 인게임 고유 명사(연어 이름 등)에만 최소한으로 독음을 표기하며, 일반 메뉴명에서는 제거합니다.
5. **표준어 기반 기술**: 가이드는 친절한 **해요체**를 사용하여 작성합니다.
