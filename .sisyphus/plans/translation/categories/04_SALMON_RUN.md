## 🤖 번역 실행 프롬프트 (Execution Prompts)

실제 번역 작업 시 각 요원(Agent)에게 부여할 '정밀 지침'입니다.

### 1. 데이터 중심 페이지용 (예: 무기 보정치)
> **프롬프트**: "이 페이지는 새먼 런 무기 수치 데이터입니다. 모든 숫자는 100% 원본을 유지하십시오. `F(Frame)` 단위는 그대로 두되 옆에 초 단위를 환산하여 병기하십시오. 일본어 부기 명칭은 반드시 `japanese-korean-terminology.json`을 참고하여 공식 한국어 명칭으로 치환하십시오. 표의 헤더는 유저가 이해하기 쉬운 명칭(예: 발사 간격, 확정 타수 등)으로 번역하십시오."

### 2. 전략 가이드용 (예: 새먼 런 해킹)
> **프롬프트**: "이 페이지는 상급자용 기술 가이드입니다. 단순히 직역하지 말고, 게임 내 실제 로직을 설명하는 '전문 기술 문서' 스타일로 번역하십시오. '寄せ(Yose)'는 '유도/끌어오기'로, 'タゲ(Tage)'는 '타겟/어그로'로 일관되게 번역하십시오. 복잡한 메커니즘은 Starlight의 `<Aside type="tip">`을 사용하여 강조하십시오."

### 3. 스테이지/적 공략용
> **프롬프트**: "스테이지 및 적 공략 정보입니다. 이미지와 텍스트의 정합성이 중요합니다. 이미지 캡션은 이미지가 설명하는 핵심 포인트를 명확히 전달하도록 번역하십시오. 적의 HP 및 데미지 데이터는 상단에 카드 형태로 요약 배치하십시오."

## 🛠️ 수집된 기술 용어 사전 (Pre-Glossary)
- **Luring (寄せ)**: 유도 / 끌어오기
- **Targeting (タゲ)**: 타겟 / 어그로
- **Delivery (納品)**: 납품
- **Recovery (回復)**: 회복 / 복구
- **Inertia Cancel (慣性キャンセル)**: 관성 캔슬
- **Wall Hanging (セミ)**: 벽 세미 / 벽 매달리기
